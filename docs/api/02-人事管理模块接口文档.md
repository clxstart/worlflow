# 人事管理模块接口文档

## 模块说明

人事管理模块包含考勤管理、请假管理、薪资管理等功能。

---

## 1. 考勤管理

### 1.1 获取考勤记录列表

**接口地址**: `GET /hr/attendance/list`

**是否认证**: 是

**权限标识**: `hr:attendance:list`

**请求参数**:

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| pageNum | int | 否 | 页码 |
| pageSize | int | 否 | 每页数量 |
| userId | long | 否 | 用户ID |
| userName | string | 否 | 用户姓名 |
| deptId | long | 否 | 部门ID |
| attendanceDate | date | 否 | 考勤日期 |
| status | string | 否 | 状态 |
| beginDate | date | 否 | 开始日期 |
| endDate | date | 否 | 结束日期 |

**响应示例**:
```json
{
  "code": 200,
  "data": {
    "total": 100,
    "rows": [
      {
        "attendanceId": 1,
        "userId": 1,
        "userName": "张三",
        "deptId": 103,
        "deptName": "研发部门",
        "attendanceDate": "2024-01-15",
        "clockInTime": "08:55:00",
        "clockOutTime": "18:30:00",
        "workHours": 9.5,
        "status": "0",
        "lateMinutes": 0,
        "earlyMinutes": 0,
        "overtimeHours": 0.5,
        "remark": null
      }
    ]
  }
}
```

### 1.2 获取考勤详情

**接口地址**: `GET /hr/attendance/{attendanceId}`

**是否认证**: 是

**权限标识**: `hr:attendance:query`

**响应示例**:
```json
{
  "code": 200,
  "data": {
    "attendanceId": 1,
    "userId": 1,
    "userName": "张三",
    "deptId": 103,
    "attendanceDate": "2024-01-15",
    "clockInTime": "08:55:00",
    "clockOutTime": "18:30:00",
    "workHours": 9.5,
    "status": "0",
    "lateMinutes": 0,
    "earlyMinutes": 0,
    "overtimeHours": 0.5,
    "remark": null
  }
}
```

### 1.3 上班打卡

**接口地址**: `POST /hr/attendance/clockIn`

**是否认证**: 是

```json
    "clockInTime": "08:55:00",
    "status": "0"

```

### 1.4 下班打卡

**接口地址**: `POST /hr/attendance/clockOut`

**是否认证**: 是

**请求参数**: 无

**响应示例**:
```json

  "data": {
```

### 1.5 补卡申请

**接口地址**: `POST /hr/attendance/supplement`

**是否认证**: 是

**权限标识**: `hr:attendance:supplement`

**请求参数**:

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| attendanceDate | date | 是 | 补卡日期 |

### 1.6 考勤统计

**接口地址**: `GET /hr/attendance/statistics`

**是否认证**: 是
