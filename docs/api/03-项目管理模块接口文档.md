# 项目管理模块接口文档

## 模块说明

项目管理模块包含项目、任务、阶段、成员、文件、日志等功能。

---

## 1. 项目管理

### 1.1 获取项目列表

**接口地址**: `GET /project/list`

**是否认证**: 是

**权限标识**: `project:list`

**请求参数**:

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| pageNum | int | 否 | 页码 |
| pageSize | int | 否 | 每页数量 |
| projectName | string | 否 | 项目名称 |
| projectCode | string | 否 | 项目编码 |
| status | string | 否 | 项目状态 |
| ownerId | long | 否 | 负责人ID |
| deptId | long | 否 | 部门ID |

**响应示例**:
```json
{
  "code": 200,
  "data": {
    "total": 50,
    "rows": [
      {
        "projectId": 1,
        "projectName": "ERP系统开发",
        "projectCode": "ERP-2024-001",
        "description": "企业资源计划系统",
        "ownerId": 1,
        "ownerName": "张三",
        "deptId": 103,
        "deptName": "研发部门",
        "status": "1",
        "startDate": "2024-01-01",
        "endDate": "2024-06-30",
        "priority": 1,
        "progress": 35,
        "createTime": "2024-01-01 00:00:00"
      }
    ]
  }
}
```

### 1.2 获取项目详情

**接口地址**: `GET /project/{projectId}`

**是否认证**: 是

**权限标识**: `project:query`

**响应示例**:
```json
{
  "code": 200,
  "data": {
    "projectId": 1,
    "projectName": "ERP系统开发",
    "projectCode": "ERP-2024-001",
    "description": "企业资源计划系统",
    "ownerId": 1,
    "ownerName": "张三",
    "deptId": 103,
    "status": "1",
    "startDate": "2024-01-01",
    "endDate": "2024-06-30",
    "priority": 1,
    "progress": 35,
    "stages": [],
    "members": [],
    "statistics": {
      "totalTasks": 100,
      "completedTasks": 35,
      "pendingTasks": 65
    }
  }
}
```

### 1.3 新增项目

**接口地址**: `POST /project`

**是否认证**: 是

**权限标识**: `project:add`

**请求参数**:

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| projectName | string | 是 | 项目名称 |
| projectCode | string | 否 | 项目编码 |
| description | string | 否 | 项目描述 |
| ownerId | long | 是 | 负责人ID |
| deptId | long | 否 | 部门ID |
| startDate | date | 否 | 开始日期 |
| endDate | date | 否 | 结束日期 |
| priority | int | 否 | 优先级 |
| remark | string | 否 | 备注 |

**响应示例**:
```json
{
  "code": 200,
  "msg": "新增成功",
  "data": {
    "projectId": 1
  }
}
```

### 1.4 修改项目

**接口地址**: `PUT /project`

**是否认证**: 是

**权限标识**: `project:edit`

**请求参数**: 同新增，需包含projectId

### 1.5 删除项目

**接口地址**: `DELETE /project/{projectIds}`

**是否认证**: 是

**权限标识**: `project:remove`

### 1.6 更新项目进度

**接口地址**: `PUT /project/progress/{projectId}`

**是否认证**: 是

**权限标识**: `project:edit`

**请求参数**:

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| progress | int | 是 | 进度百分比（0-100） |

**响应示例**:
```json
{
  "code": 200,
  "msg": "更新成功"
}
```

### 1.7 更改项目状态

**接口地址**: `PUT /project/status`

**是否认证**: 是

**权限标识**: `project:edit`

**请求参数**:

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| projectId | long | 是 | 项目ID |
| status | string | 是 | 状态（0规划 1进行中 2已完成 3已取消） |

### 1.8 获取我的项目

**接口地址**: `GET /project/my`

**是否认证**: 是

**响应**: 同列表响应

### 1.9 获取我收藏的项目

**接口地址**: `GET /project/favorite`

**是否认证**: 是

### 1.10 收藏项目

**接口地址**: `POST /project/favorite/{projectId}`

**是否认证**: 是

**响应示例**:

### 1.11 取消收藏

**接口地址**: `DELETE /project/favorite/{projectId}`

**是否认证**: 是

## 5. 文件管理

**权限标识**: `project:file:list`

| projectId | long | 否 | 项目ID |
| taskId | long | 否 | 任务ID |
| fileName | string | 否 | 文件名 |

```json
}
```
